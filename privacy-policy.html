Privacy Policy

Last updated: February 18, 2026

Overview

Habits is a Chrome extension that helps you track daily habits. Your privacy is important, and this extension is designed to keep your data entirely under your control.

Data Collection

This extension does not collect, transmit, or share any personal data. It does not make any network requests. It has no server, no analytics, and no tracking.

Data Storage

All habit data (habit names, completion history) is stored locally on your device using Chrome's built-in storage API (chrome.storage.local). This data never leaves your browser.

Data Export

You can export your data as a JSON file at any time using the Export button. This file is saved to your local device and is not uploaded anywhere.
